JustIQ - Legal awareness platform with interactive quizzes and games

It offers scenario-based challenges that test users' knowledge of laws, with multiple levels and progress tracking to encourage learning. The platform also provides simplified educational content on legal topics, helping users understand laws in a practical, accessible way. Vehicle Parking System using Python and Tkinter (GUI)

The Vehicle Parking System is a Python application with a Tkinter-based GUI designed to streamline parking management. It generates unique tokens associated with the vehicle number, representing an available parking spot. Upon entering a vehicle number, the system assigns a token for a vacant space, and if the same vehicle number is entered again, it retrieves the original token, ensuring the vehicle can be easily located and retrieved. This system simplifies the parking process, making it more efficient and organized for users.

With a minimalistic interface, this basic version is designed for straightforward, everyday calculations, offering ease of use and quick results.
